Règles de Confidentialité - Comptes
L'application Comptes est conçue pour respecter votre vie privée.
1. Collecte des données : Aucune donnée personnelle n'est collectée, transmise ou stockée sur des serveurs externes. Toutes vos informations bancaires et transactions restent exclusivement sur votre appareil.
2. Stockage : Les données sont stockées localement via la base de données Hive (NoSQL).
3. Permissions : L'application utilise les Deep Links pour recevoir des informations de MacroDroid, mais n'accède pas à vos autres applications ou données privées sans votre action.
4. Contact : Pour toute question, vous pouvez contacter le développeur via le dépôt GitHub.
Clique sur "Commit changes..." pour enregistrer.
